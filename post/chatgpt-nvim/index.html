NeoVim + AI, 敲代码直接起飞

简介

chatgpt.nvim是一个Neovim插件，它利用OpenAI的ChatGPT模型为用户提供了一个与编辑器进行对话的功能。 该插件的目标是为Neovim用户提供一个更智能、更人性化的编辑器体验。通过与ChatGPT模型的集成，用户可以通过与编辑器进行对话来执行各种编辑任务，如代码补全、语法检查、文本生成等。 chatgpt.nvim插件提供了一些命令和快捷键来与ChatGPT模型进行交互。用户可以使用命令来发送消息、获取模型的回复，并可以通过快捷键来快速切换输入和输出窗口。 该插件还支持自定义配置，用户可以根据自己的需求来调整模型的行为和设置。例如，用户可以定义自己的快捷键、设置模型的回复长度等。

chatgpt.nvim插件依赖于OpenAI的ChatGPT模型，所以我们需要一个有效的OpenAI API密钥才能使用该插件。

安装

--using Lazy.nvim
{
  "jackMort/ChatGPT.nvim",
    event = "VeryLazy",
    config = function()
      require("chatgpt").setup()
    end,
    dependencies = {
      "MunifTanjim/nui.nvim",
      "nvim-lua/plenary.nvim",
      "nvim-telescope/telescope.nvim"
    }
}

基础用法

对话

命令： 使用 gpt-3.5-turbo 模型打开交互式窗口。（也称为 ChatGPT ）

:ChatGP

和平常使用gpt一样

以对话的形式编辑代码

编辑所选文本或整个窗口

命令：

:ChatGPTEditWithInstructions

使用提示词

使用Awesome ChatGPT Prompts 的提示词

命令：

:ChatGPTActAs

运行特定的操作

ChatGPTRun [action] 命令，运行特定操作 – 有关详细列表，请参阅 actions.json 文件。可用的操作包括：

  1. grammar_correction
  2. translate
  3. keywords
  4. docstring
  5. add_tests
  6. optimize_code
  7. summarize
  8. fix_bugs
  9. explain_code
  10. roxygen_edit
  11. code_readability_analysi

更详细的使用说明请参阅 ChatGPT.nvim官方文档

总之，强烈推荐这款插件，可以大大提高你的编辑效率。可以更好的摸鱼学习和开发。
